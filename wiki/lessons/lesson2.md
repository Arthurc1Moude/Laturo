# Lesson 2: Numbers and Counting

Master the Laturo number system from 0 to 1,000,000!

## Learning Objectives

By the end of this lesson, you will be able to:
- Count from 0 to 20
- Understand the pattern for numbers 21-99
- Count by tens (10, 20, 30, etc.)
- Use large numbers (100, 1000, million)
- Use ordinal numbers (first, second, third)
- Count objects in sentences
- Tell quantities

## Cardinal Numbers 0-10

These are the foundation numbers you must memorize:

| Number | Laturo | Pronunciation | Memory Tip |
|--------|--------|---------------|------------|
| 0 | **nulo** | NOO-loh | "null" = zero |
| 1 | **uniko** | oo-NEE-koh | "unique" = one |
| 2 | **duvo** | DOO-voh | "duo" = two |
| 3 | **triko** | TREE-koh | "tri" = three |
| 4 | **katvo** | KAHT-voh | four corners |
| 5 | **penko** | PEHN-koh | pentagon = 5 sides |
| 6 | **sesvo** | SEHS-voh | six |
| 7 | **septo** | SEHP-toh | "sept" = seven |
| 8 | **okto** | OHK-toh | "octo" = eight |
| 9 | **nevo** | NEH-voh | nine |
| 10 | **deko** | DEH-koh | "deca" = ten |

### Practice Counting 0-10

Say these out loud 5 times:
```
nulo, uniko, duvo, triko, katvo, penko, sesvo, septo, okto, nevo, deko
```

## Numbers 11-19

The pattern is simple: **deko + number**

| Number | Laturo | Pattern |
|--------|--------|---------|
| 11 | **dekuno** | deko + uniko |
| 12 | **dekduvo** | deko + duvo |
| 13 | **dektriko** | deko + triko |
| 14 | **dekkatvo** | deko + katvo |
| 15 | **dekpenko** | deko + penko |
| 16 | **deksesvo** | deko + sesvo |
| 17 | **deksepto** | deko + septo |
| 18 | **dekokto** | deko + okto |
| 19 | **deknevo** | deko + nevo |

### Understanding the Pattern

Notice how the numbers combine:
- **dek** (ten) + **uno** (one) = **dekuno** (eleven)
- **dek** (ten) + **duvo** (two) = **dekduvo** (twelve)

This pattern makes numbers logical and easy to remember!

## Numbers 20-99

### Counting by Tens

| Number | Laturo | Pattern |
|--------|--------|---------|
| 20 | **dudeko** | duvo + deko |
| 30 | **trideko** | triko + deko |
| 40 | **katdeko** | katvo + deko |
| 50 | **pendeko** | penko + deko |
| 60 | **sesdeko** | sesvo + deko |
| 70 | **sepdeko** | septo + deko |
| 80 | **oktdeko** | okto + deko |
| 90 | **nevdeko** | nevo + deko |

### Numbers 21-99

For numbers between tens, combine: **tens + ones**

Examples:
- 21 = **dudeko uniko** (twenty one)
- 35 = **trideko penko** (thirty five)
- 47 = **katdeko septo** (forty seven)
- 68 = **sesdeko okto** (sixty eight)
- 99 = **nevdeko nevo** (ninety nine)

## Large Numbers

| Number | Laturo | Pronunciation |
|--------|--------|---------------|
| 100 | **sento** | SEHN-toh |
| 200 | **duvsento** | doov-SEHN-toh |
| 1,000 | **milos** | MEE-lohs |
| 2,000 | **duvmilos** | doov-MEE-lohs |
| 1,000,000 | **miliono** | mee-lee-OH-noh |
| 1,000,000,000 | **biliono** | bee-lee-OH-noh |

### Examples with Large Numbers

```
sento uniko
101 (one hundred one)

duvsento trideko penko
235 (two hundred thirty five)

milos dekpenko
1,015 (one thousand fifteen)

duvmilos katdeko duvo
2,042 (two thousand forty two)
```

## Ordinal Numbers

Ordinal numbers show position or order (first, second, third, etc.)

| Position | Laturo | English |
|----------|--------|---------|
| 1st | **unikano** | first |
| 2nd | **duvano** | second |
| 3rd | **trikano** | third |
| 4th | **katvano** | fourth |
| 5th | **penkano** | fifth |
| 6th | **sesvano** | sixth |
| 7th | **septano** | seventh |
| 8th | **oktano** | eighth |
| 9th | **nevano** | ninth |
| 10th | **dekano** | tenth |

### Using Ordinal Numbers

```
Mo estar unikano.
I am first.

To estar duvano en linealo.
You are second in line.

Esta estar trikano libro.
This is the third book.
```

## Using Numbers in Sentences

### Counting Objects

```
Mo estar uniko estudiante.
I am one student.

Esta estar duvo libros.
These are two books.

Mor estar triko amigos.
We are three friends.

Lor estar penko profesores.
They are five teachers.
```

### Asking "How Many?"

Use **vokanto** (how many):

```
Vokanto libros to estar?
How many books do you have?

Mo estar triko libros.
I have three books.

Vokanto estudiantes estar aquí?
How many students are here?

Estar dekpenko estudiantes.
There are fifteen students.
```

### Telling Age

```
Mo estar dudeko uniko jaros.
I am twenty-one years old.

To estar vokanto jaros?
How old are you?

La estar trideko penko jaros.
She is thirty-five years old.
```

## Practice Dialogues

### Dialogue 1: Counting Objects

```
Person A: Vokanto libros to estar?
          How many books do you have?

Person B: Mo estar katvo libros.
          I have four books.

Person A: Yelo vokanto mesas estar aquí?
          And how many tables are here?

Person B: Estar duvo mesas.
          There are two tables.
```

### Dialogue 2: Asking Age

```
Person A: To estar vokanto jaros?
          How old are you?

Person B: Mo estar dudeko triko jaros.
          I am twenty-three years old.

Person A: Mo estar dudeko penko jaros.
          I am twenty-five years old.
```

### Dialogue 3: Shopping

```
Person A: Vokanto kostar esta?
          How much does this cost?

Person B: Esta kostar dekpenko.
          This costs fifteen.

Person A: Mo estar desearelo duvo.
          I want two.

Person B: Duvo kostar trideko.
          Two cost thirty.
```
